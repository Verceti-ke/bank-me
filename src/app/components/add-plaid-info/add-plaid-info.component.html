1. Go to Plaid and create an account

2. Click on "Build in Development"

3. The API Keys to view transactions, authentication and balance should be provided in the dev page

4. If you have more than 5 separate bank accounts, you can fill out a 'use form' to increase the credentials limit to 100.

5. Go to the next page to add your API Keys and Secret, only you have access to this keys. Checkout the project source code on Github
